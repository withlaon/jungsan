# 라이더 정산 시스템 (JUNGSAN)

라이더 정산 시스템 프로젝트입니다.

## 구조

- `jungsan-app/` - Next.js 기반 웹 애플리케이션

## GitHub 동기화

- **원격 저장소**: https://github.com/withlaon/jungsan
- **자동 푸시**: `git commit` 시 자동으로 GitHub에 푸시됩니다 (post-commit 훅)
- 수정 후 동기화: `git add .` → `git commit -m "메시지"` → 자동 푸시

## 시작하기

```bash
cd jungsan-app
npm install
npm run dev
```
